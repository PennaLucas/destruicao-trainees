# 💥 Destruição Trainees

Bem-vindo ao repositório **Destruição Trainees**!

Este projeto foi criado para ajudar novos trainees a se familiarizarem com:

- Fork
- Pull Request
- Issues
- Navegação no GitHub

---

## 🎯 Objetivo

Explorar o GitHub enquanto interage com um projeto simples em:

- HTML
- CSS
- JavaScript
- Markdown

---

## 🚀 Atividades sugeridas

1. Faça um **Fork** deste repositório
2. Clone o repositório forkado
3. Faça uma alteração simples:
   - Mude uma cor
   - Adicione um botão
   - Crie uma nova mensagem
4. Abra uma **Issue**
5. Crie um **Pull Request**
6. Explore o repositório

---

## 💡 Ideias de melhorias

- Adicionar novos níveis
- Criar animações
- Adicionar sons
- Criar novos botões

---

## 🧪 Arquivos do projeto

| Arquivo | Descrição |
|--------|-----------|
| index.html | Estrutura da página |

| script.js | Lógica JavaScript |

## 🏆 Desafio extra

Crie uma nova funcionalidade e envie um Pull Request!

Exemplos:

- Resetar contador
- Mostrar emojis aleatórios
- Criar modo escuro
- Adicionar ranking
